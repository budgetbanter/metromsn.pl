## 📌 O projekcie

MetroMSN to nowoczesny portal internetowy dostarczający aktualne informacje, inspirujące treści oraz materiały lifestyle i rozrywkowe. Serwis umożliwia wygodne przeglądanie wiadomości, artykułów i interesujących materiałów w jednym miejscu.

➡️ Odwiedź portal: [Przejdź do MetroMSN](https://metromsn.pl)
